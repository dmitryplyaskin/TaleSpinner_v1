# Chat Basics

Ниже описан базовый пользовательский поток, подтвержденный текущим кодом.

## 1. Выбор или создание профиля

- Главный экран рендерится в `web/src/App.tsx`.
- Если профиль не выбран, UI предлагает создать новый.
- Логика профилей и чатов находится в `web/src/model/chat-core/index.ts`.

## 2. Работа с чатами и ветками

Поддерживаются операции:

- создание/удаление чатов,
- создание/активация/переименование/удаление веток,
- привязка prompt template к чату.

См. `web/src/model/chat-core/index.ts` и backend API в `server/src/api/chats.core.api.ts`.

## 3. Отправка сообщения и стриминг

- Отправка и стриминг реализованы в `web/src/model/chat-entry-parts/index.ts`.
- Используются SSE-события с обработкой `llm.stream.*`.
- Поддерживаются `send`, `continue`, `regenerate`, `abort`.

## 4. Варианты и ручное редактирование

Доступны:

- выбор варианта ответа,
- удаление варианта,
- ручное редактирование,
- soft-delete сообщений/частей.

См. `web/src/model/chat-entry-parts/index.ts`.
