# Chat Basics

Этот гайд про повседневный сценарий: как держать диалоги чистыми, управляемыми и воспроизводимыми.

## Рекомендуемый workflow

## 1. Один профиль = один контекст

Используйте отдельный профиль под отдельную роль/персонажа/сеттинг.

Почему: меньше случайного смешивания контекста между задачами.

## 2. Один сценарий = одна ветка

Для альтернативных вариантов сюжета создавайте новую ветку, а не перетирайте текущую.

Почему: проще сравнивать варианты и откатываться.

## 3. Используйте режимы генерации осознанно

- `Отправить` - стандартный ход.
- `Продолжить` - когда нужно продолжение без нового user-turn.
- `Перегенерировать` - когда не устраивает текущий вариант ассистента.
- `Остановить` - когда ответ уходит не туда и нужно прервать поток.

## 4. Поддерживайте историю в порядке

- удаляйте неудачные варианты,
- редактируйте сообщения точечно,
- скрывайте из prompt то, что не должно влиять на следующие ответы.

Это снижает шум и улучшает предсказуемость следующих генераций.

## Практический сценарий

1. Создайте чат `Сцена 1`.
2. Доведите до ключевой развилки.
3. Создайте новую ветку `Сцена 1A`.
4. В ветке тестируйте альтернативу.
5. Если ответ слабый: `Перегенерировать`.
6. Если в истории мусор: удалите/скройте лишнее.

## Частые ошибки

## Делают все в одной ветке

Итог: трудно понять, почему качество ответа упало.

## Пытаются править все вручную вместо вариантов

Итог: лишняя рутина. Быстрее сначала переключать/перегенерировать варианты.
