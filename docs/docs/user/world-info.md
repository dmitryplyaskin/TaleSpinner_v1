# World Info

World Info в TaleSpinner нужен для контролируемого добавления знаний в генерацию: лор, справки, правила мира, факты о персонажах.

## Когда включать World Info

Включайте его, если:

- модель забывает важные факты о мире,
- в одном чате много терминов/сущностей,
- нужен повторяемый результат на длинной дистанции.

## Быстрый сценарий: первая книга World Info

1. Откройте левое меню `World Info`.
2. Создайте книгу или импортируйте готовую (`.json` / `.png`).
3. Выберите книгу из списка.
4. Включите привязку к текущему чату (`Bind to chat`).
5. Сохраните настройки и отправьте тестовый запрос.

Ожидаемый результат: ответы начинают стабильнее учитывать нужный lore.

## Сильная сторона TaleSpinner

Можно привязывать книги по нескольким уровням сразу:

- `global` - общий слой для всех,
- `chat` - локально под конкретный диалог,
- `entity_profile` - под профиль,
- `persona` - под выбранную персону.

Это удобнее, чем хранить весь lore одним монолитом.

## Импорт из других экосистем

Поддерживаются форматы:

- `st_native`
- `character_book`
- `agnai`
- `risu`
- `novel`
- `auto` (автоопределение)

## Практические советы

- Сначала делайте маленькие тематические книги, не один огромный файл.
- Для локальных сюжетов используйте chat-binding, а не global.
- Держите включенными только действительно нужные книги.

## Частые проблемы

## Слишком много активных книг

Симптом: ответы становятся шумными или менее точными.

Что делать: отключите лишние привязки и сократите контекст до нужного минимума.
