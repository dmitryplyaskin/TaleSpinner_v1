# LLM Setup

Эта страница про практичную настройку без лишней ручной конфигурации.

## Минимальный рабочий путь

1. Откройте `Настройки LLM` -> `API Provider`.
2. Выберите провайдера.
3. Добавьте токен.
4. Загрузите список моделей.
5. Выберите модель.
6. Отправьте тестовое сообщение в чате.

## Что настраивается

- активный провайдер,
- активный токен,
- активная модель,
- конфиг провайдера,
- LLM пресеты.

## Зачем нужны пресеты

Пресет удобен, когда вы часто переключаетесь между режимами:

- креативный стиль,
- более строгий/детерминированный стиль,
- короткие технические ответы,
- длинные художественные ответы.

Вместо ручного выставления параметров каждый раз - одно действие `Применить пресет`.

## Модели не загружаются

Проверьте:

- токен выбран для нужного провайдера,
- базовый URL провайдера корректен,
- токен не истек и не ограничен по правам.

## Ответы слишком нестабильные

Используйте отдельный более строгий пресет и применяйте его как дефолт для вашего сценария.
