# Getting Started

## Что вы получите через 10 минут

После этого гайда у вас будет:

- созданный профиль,
- подключенный LLM-провайдер,
- первый рабочий чат,
- понимание, куда идти дальше (World Info и Operations).

## Перед началом

Нужно:

- Node.js 20+
- Yarn 1.x

## Шаг 1. Установите зависимости

```bash
yarn install:server
yarn install:web
yarn install:docs
```

## Шаг 2. Запустите приложение

```bash
yarn dev
```

По умолчанию backend работает на `http://localhost:5000`, frontend подключается к `http://localhost:5000/api`.

## Шаг 3. Создайте первый профиль

1. Откройте приложение в браузере.
2. На стартовом экране нажмите `Создать новый профиль`.
3. Выберите созданный профиль в левом меню `Профили`.

## Шаг 4. Подключите LLM

1. Откройте в левом меню `Настройки LLM`.
2. Перейдите в `API Provider`.
3. Выберите провайдера (`openrouter` или `openai_compatible`).
4. Добавьте токен.
5. Загрузите и выберите модель.

## Шаг 5. Отправьте первое сообщение

1. Откройте или создайте чат.
2. Введите сообщение и нажмите `Отправить`.
3. Проверьте, что ответ приходит потоково.

## Что читать дальше

1. `Chat Basics` - стабильный ежедневный workflow в чатах.
2. `World Info` - как управлять lore-контекстом.
3. `Operations` - как автоматизировать шаги до/после генерации.
4. `LLM Setup` - как настроить пресеты под разные задачи.
5. `TaleSpinner vs SillyTavern` - когда TaleSpinner дает преимущество.

## Частые проблемы

## Нет ответа от модели

Проверьте:

- выбран ли токен для активного провайдера,
- выбрана ли модель,
- не вернул ли провайдер ошибку лимита/авторизации.

## Интерфейс открыт, но чата нет

Это нормально: сначала выберите или создайте профиль, затем чат.
