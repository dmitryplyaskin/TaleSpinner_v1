# Основы использования Pipelines

В этом разделе мы рассмотрим, как создавать и настраивать базовые пайплайны.

## Доступ к Pipelines

Для доступа к функционалу Pipelines:

1. Откройте боковую панель приложения
2. Выберите раздел "Pipeline"

## Создание нового пайплайна

Для создания нового пайплайна:

1. Нажмите кнопку "Add Pipeline"
2. Заполните основные параметры пайплайна:
   - **Pipeline Name**: название пайплайна
   - **Pipeline Tag**: тег для использования результатов в других пайплайнах
   - **Prompt**: шаблон запроса к языковой модели

## Основные настройки пайплайна

Каждый пайплайн имеет следующие основные настройки:

### Общие параметры

- **Pipeline Name**: название пайплайна для удобства идентификации
- **Pipeline Tag**: уникальный идентификатор, который можно использовать в других пайплайнах через `{{tag}}`
- **Prompt**: текст запроса к языковой модели, может содержать переменные и теги других пайплайнов

### Параметры обработки

- **Enable Pipeline**: включение/отключение пайплайна
- **Processing**: тип обработки:

  - **Pre-processing**: выполняется перед основной генерацией
  - **Generation**: используется для генерации основного ответа
  - **Post-processing**: выполняется после основной генерации

- **Output Type**: определяет, от чьего имени будет выводиться результат:
  - **User**: от имени пользователя
  - **Assistant**: от имени ассистента
  - **System**: системное сообщение

### Параметры отображения

- **Add to Chat History**: добавлять ли результат в историю чата
- **Show to User in Chat**: показывать ли результат пользователю в интерфейсе
- **Add to Prompt**: добавлять ли результат в основной промпт при обычной генерации

## Глобальные настройки Pipelines

В верхней части интерфейса Pipelines доступны глобальные настройки:

- **Enable pipelines**: общее включение/отключение системы пайплайнов
- **Full pipeline processing**: полная замена стандартного процесса генерации на пайплайны

## Сохранение изменений

После настройки пайплайнов не забудьте нажать кнопку "Save Pipelines" для сохранения изменений.

## Запуск пайплайна

Вы можете запустить отдельный пайплайн для тестирования, нажав кнопку "Run Pipeline" (значок воспроизведения) в правом нижнем углу настроек пайплайна.
