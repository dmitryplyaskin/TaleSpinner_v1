# Расширенное использование Pipelines

В этом разделе рассматриваются более сложные аспекты работы с пайплайнами.

## Взаимодействие между пайплайнами

Одна из ключевых возможностей системы пайплайнов — это взаимодействие между ними через систему тегов.

### Использование тегов

1. Присвойте уникальный тег пайплайну в поле "Pipeline Tag"
2. Используйте этот тег в промптах других пайплайнов через синтаксис `{{tag}}`
